
# مُراقب المرتفق (Murtafiq Monitor)

تطبيق متطور مصمم لمساعدة المديرين على رصد وتحليل أداء المرتفقين بشكل يومي وعلمي.

## المميزات الرئيسية:
- **رصد الأداء:** تسجيل نقاط الالتزام والتشويش لحظياً.
- **التصنيف السلوكي:** التعرف على 21 بروفايلاً سلوكياً مختلفاً للمرتفقين.
- **الذكاء الإداري:** تقديم استراتيجيات "توازن ناش" للتعامل مع كل شخصية بفعالية.
- **الإحصائيات:** لوحة تحكم تعرض التقارير الدورية (يومية/أسبوعية/شهرية).
- **التثبيت الفوري (PWA):** إمكانية الإضافة للشاشة الرئيسية في هواتف iOS و Android.

## رسالة المشاركة (واتساب):
> **📱 تطبيق "مُراقب المرتفق"**
> أداتك الذكية لرصد أداء المرتفقين وتحليل بروفايلاتهم السلوكية يومياً بأسلوب علمي ومبتكر.
